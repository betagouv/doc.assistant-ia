## FAQ Référence

### ❓ Questions fréquentes sur les aspects légaux et techniques

**Q : Puis-je utiliser l'Assistant IA pour un usage personnel ?**
R : Non, l'Assistant IA est réservé à un **usage professionnel** dans le cadre de vos missions de service public.

**Q : Mes données peuvent-elles être partagées avec des tiers ?**
R : Non, vos données sont **strictement confinées** à l'infrastructure souveraine et ne sont jamais partagées avec des tiers commerciaux.

**Q : Que se passe-t-il en cas de violation de données ?**
R : La DINUM a mis en place des **procédures de réponse aux incidents** conformes aux exigences de l'ANSSI. Tout incident sera signalé aux autorités compétentes.

**Q : Puis-je utiliser l'Assistant IA sur mon téléphone ?**
R : Oui, l'application web est **responsive** et fonctionne sur smartphone. Une application mobile dédiée est en développement.

**Q : Comment signaler un problème de conformité ?**
R : Contactez votre DPO interne ou envoyez un email à dpd@pm.gouv.fr.

**Q : Où puis-je trouver le code source ?**
R : Le code est disponible sur [GitHub](https://github.com/suitenumerique/conversations) sous licence MIT.
