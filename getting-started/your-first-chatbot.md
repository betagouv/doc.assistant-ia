# Votre premier chat

Ce guide vous accompagne dans votre première interaction avec l'assistant IA. À la fin, vous saurez comment poser des questions, obtenir des réponses précises et utiliser au mieux votre assistant.

{% hint style="info" %}
Nous utiliserons un assistant de support client pour ce tutoriel. Les mêmes principes s'appliquent à tous les types d'assistants.
{% endhint %}

## 1. Comprendre ce que votre assistant peut faire

Avant de commencer, sachez que votre assistant IA peut :

- **Répondre à des questions** sur les produits, services et procédures de votre entreprise
- **Fournir des explications** sur des concepts complexes de manière simple
- **Vous guider** à travers des processus étape par étape
- **Rechercher des informations** dans la base de connaissances de l'entreprise
- **Vous aider à résoudre des problèmes** techniques ou opérationnels
- **Recommander des solutions** basées sur vos besoins

## 2. Accéder à votre assistant

{% stepper %}
{% step %}
#### Connectez-vous à la plateforme

Accédez à la plateforme via le lien qui vous a été fourni (sur le site web de votre entreprise, application mobile, ou intranet).

{% tabs %}
{% tab title="Site web" %}
Rendez-vous sur le site web de votre entreprise et recherchez l'icône de discussion ou le widget de l'assistant.
{% endtab %}

{% tab title="Application mobile" %}
Ouvrez l'application mobile de votre entreprise et recherchez la section d'assistance ou de chat.
{% endtab %}

{% tab title="Intégration existante" %}
Votre assistant peut déjà être intégré à des plateformes que vous utilisez (Slack, Teams, etc.).
{% endtab %}
{% endtabs %}
{% endstep %}

{% step %}
#### Authentifiez-vous si nécessaire

Selon la configuration, vous devrez peut-être vous authentifier avec :
- Votre identifiant et mot de passe d'entreprise
- Votre compte Google ou Microsoft
- Votre identifiant d'employé
{% endstep %}

{% step %}
#### Sélectionnez l'assistant

Si plusieurs assistants sont disponibles, choisissez celui qui correspond à vos besoins :
- **Support client** pour les questions sur les produits
- **Support technique** pour l'aide technique
- **Ressources humaines** pour les questions liées au personnel
- **Assistant général** pour toute autre question
{% endstep %}
{% endstepper %}

## 3. Poser votre première question

Utilisez le champ de message pour interagir avec l'assistant :

{% stepper %}
{% step %}
#### Ouvrez une nouvelle conversation

Cliquez ou appuyez sur **Nouvelle conversation** ou commencez simplement à taper.
{% endstep %}

{% step %}
#### Tapez votre question

Posez une question claire et spécifique. Exemples :
- "Quelle est la politique de congé de l'entreprise ?"
- "Comment puis-je accéder au réseau VPN ?"
- "Quels sont les avantages de notre produit phare ?"
- "Pouvez-vous m'expliquer la procédure de remboursement ?"
{% endstep %}

{% step %}
#### Envoyez votre message

Appuyez sur Entrée ou cliquez sur le bouton d'envoi. L'assistant commencera à traiter votre demande.
{% endstep %}
{% endstepper %}

{% hint style="success" %}
Vous pouvez poser des questions en français ou en anglais selon votre préférence. L'assistant comprend les deux langues.
{% endhint %}

## 4. Comprendre la réponse

L'assistant vous fournira une réponse basée sur les informations disponibles. Voici ce à quoi vous attendre :

### Format de la réponse

- **Texte clair et direct** : Réponse à votre question
- **Informations supplémentaires** : Contexte ou détails pertinents
- **Sources** : Références aux documents ou pages utilisées pour répondre
- **Suggestions** : Questions liées que vous pourriez poser

### Si l'assistant ne connaît pas la réponse

L'assistant vous informera honnêtement et peut :
- Suggérer de contacter un humain pour plus d'aide
- Vous orienter vers la bonne ressource
- Vous proposer de reformuler votre question

## 5. Continuer la conversation

Vous pouvez :

- **Poser une question de suivi** sur le même sujet
- **Demander plus de détails** sur un point spécifique
- **Changer de sujet** complètement
- **Mettre fin à la conversation** lorsque vous avez obtenu ce dont vous aviez besoin

Exemples de questions de suivi :
- "Pouvez-vous donner plus d'exemples ?"
- "Que signifie ce terme technique ?"
- "Où puis-je trouver plus d'informations ?"
- "Cela s'applique-t-il à mon cas ?"

## 6. Évaluer la réponse

À la fin de chaque réponse, vous pouvez :

- **Donner un retour positif** (👍) si la réponse était utile
- **Donner un retour négatif** (👎) si la réponse n'était pas utile
- **Laisser un commentaire** pour expliquer ce qui n'allait pas

{% hint style="info" %}
Vos retours aident à améliorer l'assistant pour tout le monde. Prenez un moment pour évaluer les réponses utiles.
{% endhint %}

## Conseils pour de meilleures réponses

1. **Soyez spécifique** : Plus votre question est précise, meilleure sera la réponse
2. **Donnez du contexte** : Expliquez brièvement pourquoi vous posez la question
3. **Posez une question à la fois** : Évitez les questions trop larges
4. **Reformulez si nécessaire** : Si la réponse n'est pas pertinente, essayez de poser la question différemment
5. **Utilisez des mots-clés** : Incluez les termes importants liés à votre question

## Exemple de conversation

**Vous** : "Quelle est la procédure pour demander un congé ?"

**Assistant** : "Pour demander un congé, vous devez remplir le formulaire RH-001 au moins 15 jours à l'avance. Vous pouvez trouver le formulaire sur l'intranet sous Ressources > Formulaires RH."

**Vous** : "Où se trouve le formulaire exactement ?"

**Assistant** : "Le formulaire RH-001 est disponible à cette adresse : [lien vers le formulaire]. Vous pouvez également y accéder via l'intranet > Ressources > Formulaires RH > Congés et absences."

**Vous** : "Merci, c'est très clair !" (👍)

## Méthodes d'accès alternatives

Selon la configuration de votre entreprise, vous pouvez accéder à l'assistant via :

- **Widget sur le site web** : Cliquez sur l'icône de discussion en bas de l'écran
- **Application mobile** : Ouvrez l'application et allez dans la section Assistance
- **Slack/Teams** : Mentionnez @AssistantIA dans un canal
- **Email** : Envoyez un email à assistant@votreentreprise.com
- **Téléphone** : Appelez le numéro dédié et interagissez par voix
