# Guide Rapide - 10 minutes pour maîtriser l'Assistant IA

Ce guide vous permet de **découvrir les fonctionnalités essentielles** de l'Assistant IA en un temps record. Suivez les étapes dans l'ordre pour une prise en main efficace.

{% hint style="success" icon="stopwatch" %}
**Temps estimé :** 10 minutes
**Niveau :** Débutant
**Prérequis :** Avoir accès à l'Assistant IA (voir [Introduction](getting-started.md))
{% endhint %}

---

## Minute 1-2 : Connexion et première conversation

### 🎯 Objectif : Se connecter et envoyer son premier message

1. **Connectez-vous** à [assistant.numerique.gouv.fr](https://assistant.numerique.gouv.fr) avec ProConnect
2. **Cliquez sur "Nouvelle conversation"** (ou appui sur **Ctrl+K**)
3. **Tapez votre première question** et envoyez :
   ```
   Bonjour ! Peux-tu te présenter brièvement ?
   ```
4. **Observez la réponse** générée par l'Assistant

✅ **Vous avez** : Créé votre première conversation avec l'Assistant IA

---

## Minute 3-4 : Poser des questions efficaces

### 🎯 Objectif : Apprendre à formuler des questions pour obtenir de meilleurs résultats

**Testez ces exemples :**

1. **Question simple :**
   ```
   Qu'est-ce que la DINUM ?
   ```

2. **Question avec contexte :**
   ```
   Je suis un agent public. Quels sont mes droits en matière de télétravail selon la réglementation 2026 ?
   ```

3. **Question structurée (méthode recommandée) :**
   ```
   Rôle : Tu es un expert en administration publique.
   Objectif : Expliquer simplement le RGPD à un collègue.
   Contexte : Mon collègue ne connaît rien au RGPD.
   Question : Peux-tu faire une explication simple avec des exemples concrets ?
   ```

**Astuce :** Plus vous donnez de **contexte** et de **précisions**, meilleures seront les réponses.

✅ **Vous avez** : Appris à formuler des questions efficaces

---

## Minute 5 : Recherche web

### 🎯 Objectif : Obtenir des informations actualisées avec la recherche internet

1. **Cliquez sur l'icône 🌐** dans la barre de saisie
2. **Posez une question sur l'actualité :**
   ```
   Quelles sont les dernières annonces du gouvernement sur la transformation numérique ?
   ```
3. **Observez** : L'Assistant effectue une recherche avant de répondre

**Alternative :** Activez la **"recherche intelligente"** dans vos paramètres pour que l'Assistant décide automatiquement quand faire une recherche.

✅ **Vous avez** : Utilisé la recherche web pour des informations à jour

---

## Minute 6 : Téléversement et analyse de documents

### 🎯 Objectif : Analyser un fichier avec l'Assistant

1. **Préparez un document** (PDF, Word, etc.) sur votre ordinateur
2. **Glissez-déposez** le fichier dans la fenêtre de conversation
   - Ou cliquez sur **📎 Joindre un fichier**
3. **Attendez** que le téléchargement se termine
4. **Posez une question sur le document :**
   ```
   Peux-tu me faire un résumé de ce document ?
   ```
   ou
   ```
   Quel est le montant total mentionné dans ce fichier ?
   ```

**Formats supportés :** PDF, JPG, PNG, DOCX, PPTX, XLSX, MD, TXT
**Taille max :** 50 Mo

✅ **Vous avez** : Téléchargé et analysé un document avec l'Assistant

---

## Minute 7 : Gestion des conversations

### 🎯 Objectif : Organiser et retrouver vos échanges

1. **Créer une nouvelle conversation** :
   - Cliquez sur **"Nouvelle conversation"**
   - Posez une question différente

2. **Revenez à la première conversation** :
   - Dans la barre latérale, cliquez sur la première conversation

3. **Renommez une conversation** :
   - Cliquez sur les **trois points (⋮)** à côté du nom
   - Sélectionnez **"Renommer"**
   - Donnez un nom explicite (ex: "RGPD - Explication")

4. **Archivez une conversation** :
   - Glissez-la vers la section **"Archivées"** dans la barre latérale

5. **Recherchez dans l'historique** :
   - Utilisez la **barre de recherche** en haut
   - Tapez un mot-clé

✅ **Vous avez** : Géré efficacement vos conversations

---

## Minute 8 : Projets et instructions communes

### 🎯 Objectif : Créer un espace de travail organisé

1. **Créer un projet** :
   - Cliquez sur **"Nouveau projet"** dans la barre latérale
   - Nom : "Rédaction administrative"
   - Instruction : "Tu es un rédacteur administratif expert. Utilise un style formel, précis et conforme aux règles de la langue française. Évite les fautes d'orthographe."
   - Cliquez sur **"Créer"**

2. **Créer une conversation dans le projet** :
   - Sélectionnez le projet "Rédaction administrative"
   - Cliquez sur **"Nouvelle conversation"**
   - Posez une question liée à la rédaction

3. **Observez** : L'Assistant adopte automatiquement le style défini dans l'instruction du projet

✅ **Vous avez** : Créé un projet avec une instruction commune

---

## Minute 9 : Changer de modèle

### 🎯 Objectif : Tester différents modèles d'IA

1. **Ouvrez les paramètres** :
   - Cliquez sur votre **avatar** en haut à droite
   - Sélectionnez **"Paramètres"**

2. **Accédez à l'onglet "Modèles"**

3. **Sélectionnez un modèle différent** :
   - Mistral Small (plus rapide)
   - Mistral Large (plus puissant)

4. **Testez avec une question** et comparez les réponses

**Note :** Certains modèles peuvent être limités selon votre administration.

✅ **Vous avez** : Changé de modèle d'IA

---

## Minute 10 : Paramètres de confidentialité

### 🎯 Objectif : Configurer vos préférences de confidentialité

1. **Allez dans Paramètres > Confidentialité**

2. **Passez en revue les options :**
   - **Autoriser l'analyse de conversation** : Activez/désactivez selon vos préférences
     - ✅ Activé : L'équipe DINUM peut accéder à vos conversations pour améliorer le service
     - ❌ Désactivé : Vos conversations restent strictement privées
   - **Autoriser la recherche intelligente** : Activez/désactivez la recherche web automatique
   - **Mesure d'audience** : Autorisez ou non la collecte de statistiques d'usage

3. **Sauvegardez** vos modifications

**Recommandation :** Désactivez l'analyse des conversations si vous travaillez avec des informations sensibles.

✅ **Vous avez** : Configuré vos paramètres de confidentialité

---

## Bonus : Astuces rapides

### 💡 5 astuces pour gagner du temps

1. **Raccourcis clavier** :
   - **Ctrl+K** : Nouvelle conversation
   - **Ctrl+F** : Rechercher dans la conversation
   - **↑** : Rappeler le dernier message

2. **Glisser-déposer** : Pour téléverser rapidement des documents

3. **Épingler une conversation** : Cliquez sur l'icône 📌 pour garder une conversation en haut de la liste

4. **Dupliquer une conversation** : Cliquez sur ⋮ > Dupliquer pour reprendre une conversation existante

5. **Exporter une conversation** : Fonctionnalité à venir pour sauvegarder vos échanges

### 🚀 3 erreurs à éviter

❌ ** Questions trop vagues** → Soyez précis
❌ ** Ne pas vérifier les informations** → Toujours croiser les sources
❌ ** Soumettre des données sensibles** → Respectez les règles de confidentialité
