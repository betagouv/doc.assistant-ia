# Introduction à l'Assistant IA

Bienvenue sur l'Assistant IA, l'outil conversationnel souverain conçu pour **tous les agents des administrations de l'État et de leurs organismes de tutelle**. Ce guide vous explique comment démarrer et tirer le meilleur parti de cette plateforme.

## Qu'est-ce que l'Assistant IA ?

L'Assistant IA est une **interface conversationnelle** basée sur l'intelligence artificielle générative qui vous permet de :

- **Poser des questions** et obtenir des réponses immédiates
- **Consulter le web** pour enrichir vos recherches avec des informations actualisées
- **Analyser des documents** (PDF, images, fichiers bureautiques) pour en extraire des informations
- **Retrouver l'historique** de vos conversations pour une continuité dans vos échanges
- **Organiser votre travail** avec des projets et des bases documentaires

{% hint style="success" icon="sparkles" %}
**Nouveau :** L'Assistant IA est désormais **accessible à tous les agents** des administrations de l'État et de leurs organismes de tutelle, sans code d'accès nécessaire.
{% endhint %}

## Prérequis

Pour utiliser l'Assistant IA, vous avez besoin de :

✅ **Un compte ProConnect** valide (votre compte professionnel)
✅ **Un navigateur web moderne** (Chrome, Firefox, Edge, Safari recommandés)
✅ **Une connexion internet** stable
✅ **Appartenir à une administration** de l'État ou un organisme sous sa tutelle

## Première connexion

### Étape 1 : Accéder à la plateforme

1. Ouvrez votre navigateur web
2. Rendez-vous sur **[assistant.numerique.gouv.fr](https://assistant.numerique.gouv.fr)**
3. Vérifiez que l'URL est correcte (https://assistant.numerique.gouv.fr)

{% hint style="warning" icon="exclamation-triangle" %}
**Sécurité :** Vérifiez toujours l'URL avant de vous connecter. Ne vous connectez **jamais** sur un site qui ne porte pas le domaine **numerique.gouv.fr**.
{% endhint %}

### Étape 2 : Se connecter avec ProConnect

1. Cliquez sur le bouton **"Se connecter avec ProConnect"**
2. Sélectionnez votre administration dans la liste
3. Entrez vos **identifiants professionnels** (ceux que vous utilisez pour accéder aux autres services de l'État)
4. Validez avec votre mot de passe

**Si vous avez oublié votre mot de passe :**
- Utilisez le lien "Mot de passe oublié ?" sur la page ProConnect
- Contactez le support informatique de votre administration

### Étape 3 : Accéder à l'interface

Une fois connecté, vous serez redirigé vers :
- **L'interface principale** si c'est votre première connexion
- **Votre dernière conversation** si vous avez déjà utilisé le service

**Félicitations !** Vous êtes maintenant connecté à l'Assistant IA. 🎉

## Créer votre première conversation

### Étape 1 : Démarrer une nouvelle conversation

1. Cliquez sur **"Nouvelle conversation"** (bouton en haut à gauche ou dans la barre latérale)
2. Ou cliquez sur le bouton **🔄** dans la barre de saisie

### Étape 2 : Poser votre première question

Dans la zone de saisie, tapez votre question. Par exemple :

```
Bonjour, peux-tu m'aider à résumer ce document ?
```

### Étape 3 : Envoyer votre message

- Appuyez sur **Entrée** (⏎) sur votre clavier
- Ou cliquez sur le bouton **📤 Envoyer**

### Étape 4 : Recevoir la réponse

L'Assistant va générer une réponse. Selon la complexité, cela peut prendre quelques secondes.

**Astuce :** Pendant la génération, vous verrez le texte apparaître mot par mot.

## Fonctionnalités de base

### Poser des questions simples

L'Assistant peut répondre à une grande variété de questions :

**Exemples :**
- "Qu'est-ce que la loi SREN ?"
- "Explique-moi le RGPD en termes simples"
- "Quelles sont les dernières actualités de l'administration publique ?"
- "Comment rédiger une note de service ?"

### Utiliser la recherche web

Pour obtenir des informations **actualisées** :

1. Cliquez sur l'icône **🌐** dans la barre de saisie
2. Posez votre question
3. L'Assistant effectuera une recherche internet avant de répondre

**Ou activez la "recherche intelligente"** dans vos paramètres pour que l'Assistant décide automatiquement quand faire une recherche.

### Télécharger un document

Pour analyser un fichier :

1. Cliquez sur l'icône **📎** ou glissez-déposez le fichier dans la fenêtre de conversation
2. Sélectionnez le fichier sur votre ordinateur
3. Attendez que le téléchargement se termine
4. Posez votre question sur le document

**Formats supportés :** PDF, images (JPG, PNG, WEBP), documents bureautiques (DOCX, PPTX, XLSX, MD)

**Taille maximale :** 10 Mo (variable selon la configuration)

## Gérer vos conversations

### Créer un nouveau projet

Un **projet** permet de regrouper plusieurs conversations avec une instruction commune.

1. Cliquez sur **"Nouveau projet"** dans la barre latérale
2. Donnez un nom à votre projet (ex: "Rédaction administrative")
3. Ajoutez une **instruction commune** (optionnel mais recommandé)
   - Exemple : "Tu es un expert en rédaction administrative. Utilise un style formel et précis."
4. Cliquez sur **"Créer"**

### Organiser vos conversations

- **Renommer** : Cliquez sur le nom d'une conversation pour le modifier
- **Archiver** : Glissez une conversation vers la section "Archivées"
- **Supprimer** : Cliquez sur les trois points (⋮) > Supprimer
- **Épingler** : Épinglez une conversation importante en haut de la liste

### Rechercher dans l'historique

1. Utilisez la **barre de recherche** en haut de la barre latérale
2. Tapez un mot-clé (recherche dans les titres des conversations)
3. Les résultats s'affichent en temps réel

## Paramètres et personnalisation

### Accéder aux paramètres

1. Cliquez sur votre **avatar** ou votre nom en haut à droite
2. Sélectionnez **"Paramètres"**

### Principaux réglages

| Catégorie | Options disponibles |
|----------|----------------------|
| **Général** | Langue, thème (clair/sombre) |
| **Confidentialité** | Autoriser l'analyse des conversations, recherche intelligente |
| **Modèles** | Modèle par défaut, modèles disponibles |
| **Notifications** | Son, notifications desktop |

### Changer de modèle

1. Allez dans **Paramètres > Modèles**
2. Sélectionnez le modèle souhaité dans la liste
3. Sauvegardez vos modifications

**Modèles disponibles :**
- Mistral Medium 3.1 (recommandé pour l'usage général)
- Mistral Small (plus rapide, moins puissant)
- Mistral Large (plus puissant, pour les tâches complexes)

## Bonnes pratiques pour bien commencer

### 1. Soyez précis dans vos questions

❌ "Parle-moi de l'administration"
✅ "Quels sont les 5 principaux textes réglementaires sur la protection des données dans l'administration publique en 2026 ?"

### 2. Fournissez du contexte

```
Contexte : Je suis responsable RH dans un ministère.
Objectif : Rédiger une note sur les nouvelles règles de télétravail.
Question : Peux-tu m'aider à structurer cette note ?
```

### 3. Itérez et affinez

- Commencez par une question large
- Affinez en fonction des réponses
- N'hésitez pas à demander des précisions

### 4. Vérifiez les informations

⚠️ **L'Assistant peut faire des erreurs.**
- Toujours croiser les informations avec des sources officielles
- Ne pas utiliser l'Assistant comme seule source pour des décisions importantes

### 5. Organisez votre travail

- Utilisez des **projets** pour regrouper vos conversations par thème
- Créez des **bases documentaires** pour vos documents fréquents
- **Archivez** les anciennes conversations
