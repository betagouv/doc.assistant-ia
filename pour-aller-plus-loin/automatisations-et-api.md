# Automatisations et API

Pour l'instant, l'assistant ne supporte pas les automatisations. Consultez la [feuille de route](feuille-de-route.md) pour connaître les intégrations futures. Pour un usage programmatique, le service [Albert API](https://ia.numerique.gouv.fr/outils-ia/albert-api/) est disponible.
